# Django 商城

<h3>使用django + flask 前後端分離</h3>

<p>
	flask負責前端，Django負責後端API，
	經由jQuery產生ajax請求，
	從後端取得資料後顯示於前端
</p>

<h3>包含以下功能：</h3>

<ol>
	<li>購物車加入</li>
	<li>結帳、產生訂單</li>
	<li>註冊、登入</li>
	<li>使用uuid + 數字隨機產生訂單號碼</li>
	<li>全選、複選結帳特定商品</li>
</ol>

<h3>使用：</h3>

<ul>
	<li>jQuery</li>
	<li>Django</li>
	<li>Flask</li>
	<li>Mysql</li>
</ul>
